==========================================================================
K210视觉识别功能说明
==========================================================================

一、功能说明
==========================================================================

功能1：颜色识别（K210_SelfLearn模式）
- 识别红、绿、黄三种颜色小球
- 实时计数每种颜色的数量
- OLED显示颜色计数

功能2：二维码识别（K210_QR模式）
- 识别二维码
- 根据二维码内容执行对应动作：
  - "goback" - 后退
  - "goahead" - 前进
  - "turnleft" - 左转
  - "turnright" - 右转
  - "buzzer" - 蜂鸣器响

二、使用方法
==========================================================================

1. 开机前旋转车轮选择模式：
   - "7.K210 QR" - 二维码识别模式
   - "11.K210 Color" - 颜色识别模式

2. 按Key1确认进入对应模式

3. 根据选择的模式烧录对应的K210程序：
   - 二维码识别模式 → 烧录 k210_qr.py
   - 颜色识别模式 → 烧录 k210_color_selflearn.py

三、K210程序文件
==========================================================================

| 文件 | 功能 |
|------|------|
| k210_color_selflearn.py | 颜色识别 |
| k210_qr.py | 二维码识别 |

四、通讯协议
==========================================================================

颜色识别：$C + R + G + Y + #
示例：$C213# = 红2绿1黄3

二维码识别：$ + 二维码内容 + #
示例：$goback# = 后退命令

五、二维码内容说明
==========================================================================

| 二维码内容 | 小车动作 |
|-----------|----------|
| goback | 后退2秒 |
| goahead | 前进2秒 |
| turnleft | 左转1秒 |
| turnright | 右转1秒 |
| buzzer | 蜂鸣器响3次 |
