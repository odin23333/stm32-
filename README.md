# STM32 平衡车项目

## 项目简介
基于 STM32F103RCT6 的两轮自平衡车项目，包含陀螺仪姿态解算、电机控制、串口通信等功能。

## 硬件平台
- 主控：STM32F103RCT6
- 陀螺仪：MPU6050
- 电机驱动：TB6612 / L298N
- 显示：OLED / LCD1602（51单片机扩展）

## 项目结构
```
32小车/
├── APP/      # 应用层代码（平衡控制、姿态解算）
├── BSP/      # 板级支持包（外设驱动）
├── CMSIS/    # CMSIS 库
├── FWLib/    # STM32 标准外设库
├── OBJ/      # 编译输出（已忽略，不提交）
└── USER/     # 用户主程序
```

## 开发环境
- Keil MDK 5
- STM32 标准库
- 编译器：ARM Compiler 6

## 编译说明
1. 打开 Keil，加载 `USER` 目录下的 `.uvprojx` 工程文件
2. 选择对应的编译器版本
3. 点击 Build 编译
4. 生成的 hex 文件在 `OBJ` 目录下

## 功能特性
- [x] 陀螺仪数据读取
- [x] 姿态解算（卡尔曼滤波/互补滤波）
- [x] 电机 PID 控制
- [x] 串口通信（与 51 单片机通信显示角度）
- [ ] 蓝牙遥控
- [ ] 寻迹功能

## 通信协议
STM32 通过串口每 100ms 发送角度数据，格式：
```
A+XXD\n
例如：A+125\n = +12.5度
      A-032\n = -3.2度
```
